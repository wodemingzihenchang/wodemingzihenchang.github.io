SQLServer升级向导

  1. 选择“SQL Server2019-KB4577194-x64”,右键，已管理员身份运行
    （SQL SERVER 2019更新版本软件可在微软官网中直接下载安装）

  2. 勾选许可条款，点击“下一步”

  3. 更新安装功能，直接点击“下一步”

  4. 检查文件，直接点击“下一步”

  5. 点击“更新”

完成更新，点击“关闭”页面

数据库管理系统安装

  1. 在安装包中找到“SSMS-Setup-CHS”，鼠标右键，以管理员身份运行

注意：安装过程中若出现如下界面，则点击“重新启动”，重启计算机/服务器，再重新操作步骤1

  1. 安装路径建议默认安装路径，点击“安装”，等待安装过程

安装成功，需要重新启动计算机/服务器，点击“重新启动”

  1. 打开SQL Server Management Studio 18(建议将SQL Server Management Studio快捷方式复制到桌面)
    默认文件路径：
    C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Microsoft SQL Server Tools 18

连接成功则表明数据库及其管理系统安装成功，如下图：

若出现SQL Server连接失败，可能原因是对应服务应用解决方案，解决方案：
打开Windows Services（服务）应用，检查SQL SERVER\MSSQLSERVER服务是否启动，若未启动，点击右键，“启动”。如下图为正常启动

  1. 安装完成后，打开SQL Server 2019配置管理器

  2. 检查SQL Server网络配置中实例协议的TCP/IP是否“已启用”，若为“禁用”状态，鼠标右键，“启用”

重启“SQL Server(MSSQLSERVER)”服务（若实例为其他名称，选择对应服务重启）
