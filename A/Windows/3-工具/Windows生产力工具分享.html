虚拟机-VMware

如果在启用了 Hyper-V 和/或 VBS 的 Windows 主机上运行 VMware Workstation 15.5.5 或更高版本，

则将使用 Windows 虚拟机监控程序平台技术启动虚拟机。如果此技术的发布版本不在特定级别，或者 Windows 主机硬件不满足某些最低要求，则 VM 将无法启动。

原因：
出现此问题的原因是系统不满足某些最低要求。

  • 支持的主机操作系统
    Windows 10 20H1 build 19041.264 或更高版本

  • 支持的工作站版本
    Workstation 15.5.5 或更高版本

  • 主机系统的处理器要求
    英特尔 Sandy Bridge
    AMD Bulldozer 或更新的 CPU

要解决此问题，请执行下列操作之一：

  • 升级主机硬件和软件以满足上述“原因”部分所述的最低要求。
  • 在 Windows 主机中禁用 Hyper-V，然后启动 VMware Workstation 并打开虚拟机电源。
