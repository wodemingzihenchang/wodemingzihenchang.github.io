Windows常见问题&方法

本文章将记录收集有关Windows的杂七杂八知识点，其他Windows问题可以查询Microsoft Community

chm 转 html

1、win+R，打开“运行”对话框输入 cmd，打开 DOS 窗口

2、输入：cd D:\abc，进入 chm 文件所在磁盘目录

3、输入：
hh -decompile 【转换后 HTML 文件的保存路径】 【CHM 文件的名称】（注意保留句式里的空格）

系统文件检查器(sfc)

使用系统文件检查器工具修复丢失或损坏的系统文件 - Microsoft 支持

操作：按“Windows 徽标键+X”，动“Windows PowerShell( 管理员 )”，依次执行以下命令

glTF

glTF 俗称“3D 界的 jpeg”。glTF 2.0 原生支持 metallic workflow 的 PBR 材质

glTF 格式其实是一个文件夹，包含一个基于 json 的属性文件，一个二进制文件用来放顶点数据，以及各种图片作为纹理。这样解析起来就很简单。

资源网站：http://glbxz.com

在线查看 gltf：http://gltf.bimant.com/

系统

系统信息

在【“开始”菜单-Windows 管理工具-“系统信息”】打开工具。或“运行”对话框。键入msinfo32打开系统信息工具。

异常

文件夹或文件名称底部有下划线，是【文件夹选项】，勾选了单击打开的选项。

直接拖动文件打不开

大多数人安装好win10 后，很多软件的拖拽文件拖到软件上直接打开的功能失效，这时必须要用软件里的打开功能，这是非常麻烦的。经过测试有两种方法可以解决。

方法一：

按Windows键+R，打开“运行”对话框：输入regedit，回车或确定。依次找到以下键值： 右键“EnableLUA 选“修改”。把值改成0，确定。然后重启电脑

方法二：

按下“Win+R”，再出现的运行命令中输入“gpedit.msc”后回车。

在本地组策略编辑器界面，依次展开左侧面板：用户配置-〉管理模板-〉windows组件-〉任务计划程序，点击选中“任务计划程序”，在右侧窗口找到打开“禁止拖放”。

在禁止拖放界面，点击选择“未配置”，或者“已禁用”后，点击确定后，重启电脑即可。

错误代码

0x80073712

1、右键【开始菜单-命令提示符(管理员)】

2、打开管理员命令提示符，输入下面两个指令(一次一个)

　　01DISM.exe /Online /Cleanup-image /Scanhealth

　　01DISM.exe /Online /Cleanup-image /Restorehealth

0xc0000001

你的设备遇到问题，需要重启。终止代码：DRIVER_UNLOADER_WITHOUT_CANCELLING_PENDING_OPERATIONS

失败操作：nvpcf.sys

方法

该问题出现在安装问题程序导致的电脑启动异常问题。

可以在开机时，按F8进入“安全模式”。在安全模式下卸载问题程序即可。

当卸载程序异常时，可以开了使用“MicrosoftProgram_Install_and_Uninstall.meta”微软卸载助手，进行卸载。
