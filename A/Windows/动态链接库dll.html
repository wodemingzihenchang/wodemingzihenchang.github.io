动态链接库dll

注册dll

  1. 以管理员身份打开 Windows 命令提示符。
  1. 将正常文件复制到 C:\Windows\System32
  2. 用管理员启动 cmd，使用 regsvr32 注册.dll 文件。

（注意：如果dll注册文件不在C:\Windows\System32位置，则需执行 【cd】 命令跳转到在文件所在位置，再使用 regsvr32 注册.dll 文件）

卸载dll

1.以管理员身份打开 Windows 命令提示符。
2.在命令提示符中输入：cd [DLL 路径]
3.接着输入：regsvr32 xxx.dll -u。

Q&A

XXX.dll文件报错

XXX.dll 文件报错，无法载入 dll 文件

原因：一般会是 dll 注册问题，或是文件丢失或是被病毒杀毒损坏等。将.dll 文件找回，并放置在正常位置即可。如下：

  • exe 所在目录
  • windows 系统目录
  • windows 目录
  • 当前目录
  • PATH 指定的目录
