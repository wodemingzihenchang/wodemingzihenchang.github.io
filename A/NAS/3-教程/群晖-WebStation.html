群晖-WebStation

Web Station 是一款强大的 Web 服务应用，为用户提供了简单而强大的搭建和管理服务，可视化的操作可以让我们轻松管理网站以及运行环境。今天就来说说如何通过 Web Station 搭建自己的静态导航网站

下载套件

在群晖的【套件中心】搜索 Web Station 直接进行安装即可

打开 File Station，也会发现多出了 web 和 web_packages 文件夹，这两个文件夹都是Web Station 套件创建的

新建

回到 Web Station 套件，在网页服务选项下新增一个服务

配置

可以直接打开 web 文件夹下的网站目录修改里面的 index.html 文件

浏览

现在可以通过【http://内网IP:端口】访问到导航站了。
