I版本与非I版本区别

SOLIDWORKS API中的方法、属性和对象（接口）有两个版本可供选择：

  • 以I开头的版本（例如ISldWorks、IModelDoc2、IAnnotation、ISldWorks::IActiveDoc）
  • 不以I开头的版本（例如SldWorks、ModelDoc2、Annotation、SldWorks::ActiveDoc）

这两个版本对应的是同一个对象或方法。主要区别如下：

  • I版本的方法不会公开事件
