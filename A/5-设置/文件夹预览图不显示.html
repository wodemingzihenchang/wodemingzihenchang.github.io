文件预览图不显示

SOLIDWORKS 缩略图能够便于识别想要打开的模型，并会有图形显示如下图：

方法1：检查设置

1.首先确保 Windows 文件夹选项设置，显示的是缩略图而不是图标。所以，不勾选“始终显示图标，从不显示缩略图”，win7 和 win10 的选项位置略有不同，下图是 win7 的截图。

2.检查 windows 的性能选项，找到视觉效果标签栏，勾选“显示缩略图，而不是显示图标”：

3.当 Windows 设定完成后，若还是不显示缩略图，需检查 SOLIDWORKS 系统选项

方法2：修复安装

选项若都没有问题，可再检查 SOLIDWORKS 的文档是否是旧版本，例如电脑上先安装高版本再安装低版本SW，那会有可能导致缩略图不显示的问题(如图部分文件显示)。

此时需要用高版本修复安装，修复时注意关闭杀毒软件并且有管理员权限。

方法3：选择默认启动

SWLauncher启动方法：文件的预览和文件类型图标是由swshellLauncher.exe程序作用。因此，如果SW文件的启动方式被修改了那么。文件的图标以及预览图也会异常。如图

方法：则需要SW文件【右键-打开方式-选择其他应用】修改默认启动程序。

这个默认启动也可以在【window设置-默认应用-按文件类型指导默认应用】里设置，这点可以参考：

  1. 在”开始”菜单上，选择”设置> 应用> **默认应用”**。
  2. 选择你想要设置的默认程序，然后选择该应用。 你还可以在 Microsoft Store 中获取新的应用。 必须先安装应用，然后才可以将其设置为默认应用。
  3. 你可能希望你的.pdf、电子邮件或音乐使用 Microsoft 提供的应用自动打开。 按文件类型选择默认应用。 向下滚动并选择 **”按文件类型选择默认应用”**。
