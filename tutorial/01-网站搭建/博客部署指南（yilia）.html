Markdown写博客技巧,主要是针对主题hexo-yilia的一些写作技巧汇结。

hexo

创建新的布局

hexo new [layout] <title>
hexo new post 文章名
hexo new page 页面名
hexo new draft 草稿文章

hexo_config.yml设置

部署

url: https://用户名.github.io/
permalink: /post/:title.html

deploy:
-type: git
-repository: https://github.com/用户名/用户名.github.io.git
-branch: main （原master）

yilia_config.yml设置

menu: 归档: /archives/index.html

hexo 写作

文章属性

文章属性（Front-matter）用于指定个别文件的变量

layout (T/F)
title: 标题
date: 建立日期
update: 更新日期
comments: 评论(T/F)
tags: 标签
categoreies: 分类
permalink: 覆盖文章网址
toc: 开启目录(T/F)

显示文章摘要

当主页只需要显示少部分文章内容时，可在文章中插入“<!–more–>”隐藏后面的内容。
（备注：解决出现两个阅读全文时，可在yilia_config\注释掉# excerpt_link: more）

修改代码快背景颜色和字体颜色

修改文件位置：themes\yilia\source\main.ocf68a.css

  1. 代码块背景色：.article-entry .highlight
  2. 代码块字体颜色.article-entry .highlight .line
